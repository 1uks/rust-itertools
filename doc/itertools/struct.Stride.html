Similar to the slice iterator, but with a certain number of steps (stride) skipped per iteration.

Does not support zero-sized A.

Iterator element type is &'a A

impl<'a, A> Stride<'a, A>

fn from_slice(xs: &'a [A], step: int) -> Stride<'a, A>

Create Stride iterator from a slice and the element step count.

If step is negative, start from the back.

Example

use itertools::Stride;

let xs = [0i, 1, 2, 3, 4, 5];

let mut front = Stride::from_slice(xs.as_slice(), 2);
assert_eq!(front[0], 0);
assert_eq!(front[1], 2);

let mut back = Stride::from_slice(xs.as_slice(), -2);
assert_eq!(back[0], 5);
assert_eq!(back[1], 3);
